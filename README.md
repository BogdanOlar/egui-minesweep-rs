
# minesweep-rs

A rust implementation of the popular game, using the [egui](https://github.com/emilk/egui) library.

## Screenshot
<img src=".github/Screenshot.png" alt="Minesweep-Rs" />

## Build & run

### Desktop

```bash
git clone https://github.com/BogdanOlar/minesweep-rs.git
cd minesweep-rs/
cargo run --release
```
